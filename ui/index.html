corporate punk

self-hosted image placeholder/server

Throw a bunch of images inside a folder and run the server

usage

parameter description example
resize Resizes the final image by given width and height: ?resize=200x250 ?resize=400x400
crop Crops the final image by given width and height: ?crop=200x250 ?crop=200x250
quality Sets the image quality: [1-100] ?quality=25
image Select an specific image, if not present, a random image will be returned ?image=calaveras.jpg
List image folder Clear cache
